== Description ==
1、抓取采集网站固定内容并保存到Wordpress中。
2、自动去除文章中的内链（需要Auto save image 插件）。
3、自动保存目标站的图片（需要Auto save image 插件）。
4、可以替换目标站关键词。
定时自动采集功能待开发中。
